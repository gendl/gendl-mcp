Parameter: *Hash-Transforms?*

*HASH-TRANSFORMS?* boolean
Controls whether the transformations of points from three-dimensional world coordinates to two-dimensional display coordinates are cached in memory. Set this to NIL to save memory at the expense of some execution speed. Note: in the current GDL release, transform caching is temporarily disabled, so this parameter will have no effect. Defaults to T.
