Parameter: *Remember-Previous-Slot-Values?*

*REMEMBER-PREVIOUS-SLOT-VALUES?* boolean
Determines whether the system keeps track of previous slot values (accessible with previous-value function) after bashings are done. Leave this set to nil to improve memory performance.
