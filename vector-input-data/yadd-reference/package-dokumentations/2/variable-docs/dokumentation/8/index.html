Parameter: *Compile-For-Dgdl?*

*COMPILE-FOR-DGDL?* boolean
Determines whether global methods are defined during compilation to allow calling any message on a gdl remote-object. This functionality is not available in the base by itself, it requires the :gwl system as well. Defaults to nil.
