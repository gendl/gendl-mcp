Parameter: *Allow-Nil-List-Of-Numbers?*

*ALLOW-NIL-LIST-OF-NUMBERS?* boolean
If set to t, allows list-of-numbers function to return nil instead of error when num1 is greater than num2 with positive increment or vice-versa. Default is nil.
