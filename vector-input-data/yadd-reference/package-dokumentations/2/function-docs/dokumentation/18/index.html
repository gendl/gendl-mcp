Macro: Define-Object-Amendment

DEFINE-OBJECT-AMENDMENT supplements or alters an existing gdl object definition

Syntax is similar to that for define-object. Note that there is currently no way to undefine messages defined with this macro, other than redefining the original object or restarting the GDL session. Support for surgically removing messages will be added in a future GenDL release.
