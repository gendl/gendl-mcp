Macro: With-Error-Handling

WITH-ERROR-HANDLING [macro]

Wraps the body of code with error-trapping and system timeout. A warning is given if an error condition occurs with body.

keyword arguments:
timeout Timeout in Seconds, Default Value: 2
timeout-body Body of code to evaluate if timeout occurs, Default Value: NIL
Default is to print a warning and return nil
rest arguments:
body Body of code to be wrapped
