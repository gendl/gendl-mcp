Function: Find-Messages-Used-By

FIND-MESSAGES-USED-BY list of pairs of instance/keyword

This returns the list of direct dependants of a given message in a given instance. Note that this is not recursive; if you want to generate a tree, then you have to call this recursively yourself. If you want an easy way to remember the meaning of dependant and dependency: You have a dependency on caffeine. Your children are your dependants.
