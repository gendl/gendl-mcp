Function: Point-On-Vector?

POINT-ON-VECTOR? boolean

Determines whether or not the unknown-point lies on the ray specified by the vector pointing from first-point to second-point, within tolerance.

arguments:
first-point first point of vector
second-point second point of vector
unknown-point point in question
keyword arguments:
tolerance Tolerance for vectors to be considered same-direction, Default Value: *ZERO-EPSILON*
