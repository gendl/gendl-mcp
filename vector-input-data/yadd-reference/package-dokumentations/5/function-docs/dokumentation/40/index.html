Macro: Merge-Display-Controls

MERGE-DISPLAY-CONTROLS plist of display controls

This macro "merges" the given display controls list with that coming as a trickle-down slot from the parent. It will replace any common keys and add any new keys.

arguments:
display-controls Plist
The new display controls to be merged with the defaults from the parent
