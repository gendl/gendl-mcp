Function: Point-On-Plane?

POINT-ON-PLANE? boolean

Determines whether or not the 3d-point lies on the plane specified by plane-point and plane-normal, within tolerance.

arguments:
3d-point Point in question
plane-point point on the known plane
plane-normal normal to the known plane
keyword arguments:
tolerance Tolerance for points to be considered coincident, Default Value: *ZERO-EPSILON*
