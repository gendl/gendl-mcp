Function: Gwl:Publish-String-Content

PUBLISH-STRING-CONTENT string (representing a url path)

Publishes given url to respond with text content as specified by given string.

arguments:
url String
The url to be published
string String
The content to be emitted when the url is requested
rest arguments:
publish-args plist
Arguments to be passed on to publish function, e.g. :content-type
