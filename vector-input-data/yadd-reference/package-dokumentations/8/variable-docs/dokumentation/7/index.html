Parameter: Gwl:*Jump-To-Toplevel-On-Set-Self?*

*JUMP-TO-TOPLEVEL-ON-SET-SELF?* boolean
Set this to non-nil if you want to jump to the toplevel interaction buffer in emacs when setSelf link is pressed and *break-on-set-self?* is nil. Defaults to t.
