Function: Transform-And-Translate-Point

TRANSFORM-AND-TRANSLATE-POINT 3d-point

Returns the product of vector and transform, translated by (i.e. added to) trans-vector.

arguments:
vector 3D Vector
transform 3x3 Rotation Matrix
trans-vector 3D Vector
examples:
(let ((transform (make-transform '((0.0 0.0 1.0)
                                    (0.0 1.0 0.0)
                                    (1.0 0.0 0.0))))
       (v (make-vector 1.0 2.0 3.0))
       (t-v (make-vector 3.0 0.0 0.0)))
   (transform-and-translate-point v transform t-v))
 ---> #(6.0 2.0 1.0)
