Function: Rotate-Point-D

ROTATE-POINT-D 3d point

Returns the 3D Point resulting from rotating point about center in the plane defined by normal. The rotation can specified either by an arc length (arc-length) or an angle in degrees (angle). A second value is returned, which is the resulting angle of rotation in degrees (this is of possible use if arc-length is used to specify the rotation).

arguments:
point 3D Point
center 3D Point
normal 3D Vector
keyword arguments:
arc-length Number, Default Value: NIL
angle Number, Default Value: NIL
