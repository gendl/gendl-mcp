Macro: Translate

TRANSLATE [macro] 3d point

Within the context of a GDL object definition (i.e. a define-object), translate origin by any number of offsets.

arguments:
origin 3D Point
rest arguments:
offsets Plist consisting of direction keywords and numbers
A direction keyword can be one of:
  • :top (or :up)
  • :bottom (or :down)
  • :left
  • :right
  • :front
  • :rear (or :back)
