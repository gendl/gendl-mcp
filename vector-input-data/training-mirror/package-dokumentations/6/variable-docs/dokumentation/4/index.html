Parameter: Geom-Base::*Zero-Vector-Checking?*

*ZERO-VECTOR-CHECKING?* boolean
Indicates whether GDL should perform checks for zero-length vectors where nonzero-length vectors are expected. Defaults to T. Setting to NIL will increase execution speed marginally, at the cost of decreased safety. Modify the value of this parameter at your own risk.
