Function: Universal-Time-From-Iso-8601

UNIVERSAL-TIME-FROM-ISO-8601 integer representing common lisp universal time

Returns the universal time from a date formatted as an iso-8601 date, optionally with time, e.g. 2012-07-08 or 2012-07-08T13:33 or 2012-07-08T13:33:00
